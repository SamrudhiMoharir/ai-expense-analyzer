# **AI-Based Expense Analyzer** 

### Overview:
A backend system that categorizes financial expenses using Machine Learning and provides monthly analytics and saving recommendations via REST APIs.

### Features:
ML-powered expense categorization (Naive Bayes)
REST API built with Flask
SQLite database for persistent storage
Monthly expense summary by category
Rule-based saving suggestions
Postman-tested endpoints

### Tech Stack:
Python
Flask
Scikit-learn
SQLite
Joblib

### Architecture:
ML layer handles categorization
Service layer handles business logic
API layer handles HTTP requests

### Learning Outcomes:
Built and deployed an ML-powered backend system
Understood ML limitations and system design
Applied clean architecture and OOP principles
